Button & Input Verknüpfung

Resultat

Erklärung zu der Funktion, schrittweise:

1. Button erstellen mit dem tag: <button>. 
<button>MyButton</button>
2. Um dem Button bisschen Style zu verleihen, benötigt man dafür: <style>
<style>

.buttonStyle {

  background: rgb(39, 39, 39);
  border-radius: 10px;
  color:white;
  padding:10px;
  margin:10px;
  width:20%;
}

</style>
3. Damit der Button den Style von ".buttonStyle" bekommt, brauchen wir das class-Attribut.

  <button class="buttonStyle">MyButton</button>

  <style>
  .buttonStyle {

    background: rgb(39, 39, 39);
    border-radius: 10px;
    color:white;
    padding:10px;
    margin:10px;
    width:20%;
  }
  
  </style>
4. Input erstellen mit dem tag: <input>.

  <button class="buttonStyle">MyButton</button>

  <input></input>

16. Zu guter Letzt soll der Button sein Hintergrund ändern, wenn man mit dem Cursor über den Button hält.
Das funktioniert, indem man bei <style> ein className hinzufügt mit der Pseudoklasse ":hover", also "schweben". 

    <button class="buttonStyle" id="buttonID">MyButton</button>
    <input class="textboxStyle" id="inputID" readonly placeholder="Click on MyButton..."></input>
    
    <script>
  let deinText = "Das hier ist ein Beispieltext.";
  
  document.getElementById('buttonID').addEventListener('click', function(){
  
    alert("Die Ausgabe von dem Input lautet: " + deinText);
  
    document.getElementById('inputID').value = deinText;
  });
  document.getElementById('inputID').addEventListener('click', function(){
  
    document.getElementById('inputID').value = "";
  });
  
    </script>

    <style>

  .buttonStyle {

    background: rgb(39, 39, 39);
    border-radius: 10px;
    border:none;
    color:white;
    padding:10px;
    margin:10px;
    width:20%;
  }
  .buttonStyle:hover{

    background:gray;
  }
  .textboxStyle {

  background: rgb(39, 39, 39);
  border-radius: 10px;
  border:1px #888 solid;
  color:white;
  padding:10px;
  margin:10px;
  width:20%;
}

</style>
