Die erste Webseite erstellen mit HTML.

Wie erstelle ich eine Webseite?

1. Öffnen Sie eine App, die HTML unterstützt. Zum Beispiel Editor, Notepad ++, Visual Studio Code, Replit, ...
2. Schreiben Sie den folgenden Code in ihre App: 


        <html>

        <title>Meine erste Webseite</title>
        <p>Meine erste Webseite</p>

        </html>


3. Geben Sie ihrer Webseite bisschen Hintergrundfarbe mit der CSS-Eigenschaft "background" und dem style-Attribut. 


        <html style="background:gray">

        <title>Meine erste Webseite</title>
        <p>Meine erste Webseite</p>

        </html>


4. Um den Text zu vergrößern und es in die Mitte zu bringen, benötigen Sie die CSS-Eigenschaften "font-size" & "text-align".
(Die Standard-Schriftgröße beträgt 16px.) 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    </html>


5. Fügen Sie einen Text hinzu mit dem tag <div>. 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    </html>


6. Fügen Sie einen Button hinzu mit Hilfe von <button>. 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    <button>Webseite schließen</button>

    </html>


7. Damit Sie die Webseite schließen können, benötigen Sie JS (JavaScript).
8. Dafür müssen Sie zuerst eine ID angeben für ihr Button mit dem id-Attribut. 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    <button id="webseiteSchließen">Webseite schließen</button>

    </html>


9. Fügen Sie einen tag namens <script> hinzu. 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    <button id="webseiteSchließen">Webseite schließen</button>

    <script>



    </script>

    </html>


10. Jetzt kommt die Methode document.getElementById() ins Spiel.

Das "getElementById" greift direkt auf ein Element zu, das durch sein id-Attribut eindeutig identifiziert ist.
Achten Sie auf Groß - und Kleinschreibung, da die ID übereinstimmen muss.

Außerdem müssen Sie die Methode "addEventListener" dazu schreiben, damit der Button weiß, auf welche Events was ausführt werden soll.
Beispiel-Events sind: click, load, change, resize, mouseup, mousedown, ... 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    <button id="webseiteSchließen">Webseite schließen</button>

    <script>

document.getElementById('webseiteSchließen').addEventListener('click', function(){

    
})

    </script>

    </html>


11. Fügen Sie in die Methode diese Funktion hinzu: window.close(); 


    <html style="background:gray;font-size:30px;text-align:center">

    <title>Meine erste Webseite</title>
    <p>Meine erste Webseite</p>

    <div style="background:white;font-size:20px">
    Drücken Sie auf den Button unten, um diese Webseite zu schließen.
    </div>

    <button id="webseiteSchließen">Webseite schließen</button>

    <script>

document.getElementById('webseiteSchließen').addEventListener('click', function(){

    window.close();
})

    </script>

    </html>
